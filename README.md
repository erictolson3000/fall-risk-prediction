# Predicting Fall Risk in Older Adults

## Overview

This capstone project applies machine learning techniques to predict mobility limitations associated with fall risk in older adults using data from the CDC Behavioral Risk Factor Surveillance System (BRFSS). The project demonstrates the complete data analytics workflow from data preparation through model evaluation.

## Business Problem

Falls are one of the leading causes of injury among older adults. Predictive analytics can help identify individuals at greater risk, allowing healthcare organizations to prioritize preventive interventions and improve patient outcomes.

## Dataset

- Source: CDC Behavioral Risk Factor Surveillance System (BRFSS) 2024
- Records: 10,511
- Target Variable: Difficulty Walking (DIFFWALK)

## Tools & Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Models Evaluated

- Logistic Regression
- Decision Tree
- Random Forest

## Results

Logistic Regression achieved the highest overall accuracy while Random Forest provided additional insight into the variables most strongly associated with mobility limitations.
